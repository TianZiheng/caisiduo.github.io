Hexo主题：LightOne

博客终于能有点干货了QwO，好啦切入正题，这款我发布的第一个主题，它的名字你们也看到了，就叫LightOne，它是基于Hexo官方Light主题修改而成的。其身由钛金打造而成，我给它取名为8843，

这款主题以简洁为主，看到我的博客就知道了吧qwq，至于把这个主题发布出来的原因嘛，不是有人找我要，只是最近博客产量有点低了qwq。

项目地址：LightOne

特色

  • 单栏
  • 简洁
  • Gallery Post
  • Link Post
  • Fancybox
Gallery Post
Link Post

配置

  • menu - 设置菜单
  • excerpt_link - 在摘录的文章底部添加“ReadMore”链接文本
  • addthis - 在文章底部的分享按钮 (Powered by [AddThis])
    • enable - 启用分享按钮
    • pubid - 你的pudid码 [AddThis]
    • facebook - 启用 Facebook 按钮
    • twitter - 启用 Twitter 按钮
    • google - 启用 Google+ 按钮
    • pinterest - 启用 Pinterest 按钮
  • fancybox - 启用 [Fancybox]
  • google_analytics - 谷歌分析ID
  • rss - 订阅RSS的链接

PS: 如果你想去除脚底版权也不是问题，我只是希望你可以给我的博客带来点访问量，仅此而已。
