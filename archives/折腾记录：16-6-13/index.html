折腾记录：16/6/13

此次折腾耗时 [2] 天，折腾内容就是Bitmoe的 Hexo便携版

官方文档：

说明：

  1. 无需安装 Nodejs 和 Git ； / The installations of Nodejs and Git are not needed;

  2. 添加了一些适合适合毫无基础的用户的脚本； / Some Shells added for computer illiterate;
    注意：配置Github部署 脚本在使用时，会将 用户名.github.io 仓库中的内容以Hexo生成的文件覆盖！

  3. 更多信息请参考 Hexo 便携版 项目主页； / For more information, please refer to PortableHexo Project Homepage.

版本

  • Git: 2.7.4
  • Nodejs: 4.4.0
  • Npm: 3.8.3
  • Hexo: 3.2

使用心得

已经被折腾疯
这个便携版呢，其实就只是把各种配置文件都塞到了文件夹里，然而我做不到
把文件解压后有以下文件

blog
support
配置Github部署.bat
配置基本信息.bat
启动命令行.bat
新建文章.bat
渲染并本地测试.bat
渲染并部署.bat
重置配置文件.bat

blog 文件夹就是Hexo的根目录文件，里面的东西用过 Hexo 的都知道了吧。
support 文件夹就是系统配置文件，比如Git、Nodejs。
剩下的.bat文件看名字就应该都懂了吧

  1. 首先打开 配置Github部署.bat 配置你的Github。根据提示来填写，最后查看你的Github仓库看到有更新就行了。
  2. 接着打开 配置基本信息.bat 配置基本信息，其实就是修改Hexo的根目录 config.yml 还有 support 下的 config.yml
  3. 当配置完毕后就可以开始玩耍你的 Hexo便携版 惹！

失败次数：N 次
失败原因：不了解配置
解决方法：系统E盘测试→测试成功→转到U盘→继续测试→失败→修改→成功
最终代码：None
